**Автор:** Холост

## Описание уязвимости

**CVE‑2023‑28676** — уязвимость внедрения команд в веб‑приложении.

**Суть уязвимости:** приложение выполняет пользовательский ввод без фильтрации, что позволяет внедрять дополнительные команды через разделители (`;`, `|`).

**Потенциальная угроза:** выполнение произвольных команд на сервере (чтение файлов, запуск процессов и т. д.).

## PoC‑скрипт

Скрипт `poc_cve_2023_28676.py` имитирует отправку вредоносного запроса, содержащего команду `id`.

## Инструкция по запуску

### Требования
* Python 3.6+
* Библиотека `requests`

Запуск скрипта
bash
python poc_cve_2023_28676.py
Шаг 1. Установка и настройка инструментов
Установка Python
Перейдите на официальный сайт python.org.

Скачайте последнюю версию Python для вашей ОС.

При установке отметьте опцию «Add Python to PATH».

Проверьте установку: откройте командную строку и выполните:

python --version
или

python3 --version
Установка VSCode
Скачайте VSCode с официального сайта code.visualstudio.com.

Установите редактор, следуя инструкциям установщика.

Настройка VSCode для работы с Python
Откройте VSCode.

Нажмите Ctrl+Shift+X (или Cmd+Shift+X на macOS) для открытия панели расширений.

Найдите и установите расширение «Python» от Microsoft.

Выберите интерпретатор Python:

нажмите Ctrl+Shift+P для открытия командной палитры;

Создайте виртуальное окружение:

bash

Активируйте виртуальное окружение:
